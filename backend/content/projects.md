## Agentic RAG System with Neo4j

Repository: https://github.com/PabbisettySivaKumar/GenAi-End_to_End

This project is an end-to-end Retrieval-Augmented Generation system integrating Ollama LLMs, Neo4j Vector Database, Langfuse, Streamlit, and modular backend services.

Capabilities:
- Upload and process PDFs.
- Chunk and embed documents.
- Store and retrieve embeddings through Neo4j.
- Query documents through retrieval and answer generation.
- Track prompts and pipeline behavior through Langfuse.
- Provide a Streamlit user interface for interaction.

Relevant technologies:
- Python
- LangChain
- Neo4j Vector DB
- Ollama
- Langfuse
- Streamlit
- RAG

## AI-Powered Reddit Analytics Pipeline

Source: Resume.

This project automates ingestion and analysis of Reddit posts and nested comments. It uses Gemini to extract sentiment, identify purchase intent, detect complaint signals, and generate context-aware rewrite suggestions.

Capabilities:
- Fetches Reddit posts and nested comments through the Reddit API.
- Processes social discussion data at scale.
- Uses Gemini for sentiment classification.
- Detects high-intent purchase and complaint signals.
- Evaluates peer-to-peer answer quality.
- Stores raw and structured AI outputs in MongoDB.

Relevant technologies:
- Python
- Reddit API
- Gemini
- MongoDB
- Sentiment analysis
- Purchase-intent detection

## TawasolPay Risk Assistant

Repository: https://github.com/PabbisettySivaKumar/TAWASOLPAY-RISK-ASSISTANT

The TawasolPay Risk Assistant is an AI-powered cyber risk prioritization project for a fictional payments company. It ingests daily security data, scores vulnerabilities across multiple signals, retrieves grounded NIST 800-53 guidance through RAG, and presents top risks in a Streamlit UI.

Public demo links from the repository:
- Frontend: https://itspsk-tawasolpay-frontend.hf.space
- Backend: https://itspsk-tawasolpay-backend.hf.space
- Backend docs: https://itspsk-tawasolpay-backend.hf.space/docs

Capabilities:
- Ranks security risks.
- Retrieves NIST guidance through RAG.
- Uses a FastAPI backend.
- Provides a Streamlit frontend.
- Generates LLM explanations for risk findings.

Relevant technologies:
- Python
- FastAPI
- Streamlit
- RAG
- Cyber risk analysis
- Docker

## Leave Management System
